# Friends Radar - AI Agent Instructions

## Project Overview
Friends Radar is a Vue.js + TypeScript application built with Vite.

## Tech Stack
- **Frontend**: Vue 3 + TypeScript
- **Build Tool**: Vite
- **Package Manager**: npm

## Commands
```bash
npm run dev      # Start development server
npm run build    # Build for production
npm run preview  # Preview production build
npm run lint     # Run linter (if configured)
npm run test     # Run tests (if configured)
```

## Project Structure
```
friends-radar/
├── src/
│   ├── components/    # Vue components
│   ├── assets/        # Static assets
│   ├── App.vue        # Root component
│   └── main.ts        # Entry point
├── public/            # Public static files
├── specs/             # Feature specifications
├── prd.json           # Product requirements (Ralph)
├── progress.txt       # Ralph iteration learnings
└── PROMPT_build.md    # Build mode instructions
```

## Development Guidelines
1. Write TypeScript with strict types
2. Use Vue 3 Composition API
3. Keep components small and focused
4. Write tests for critical functionality
5. Run `npm run build` to verify no errors before committing
